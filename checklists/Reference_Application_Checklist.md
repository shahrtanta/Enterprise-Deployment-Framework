# Reference Application Checklist

- [ ] Builds in Release.
- [ ] Runs on loopback.
- [ ] `/health` returns success.
- [ ] `/Setup` works before database configuration.
- [ ] Static setup assets are local.
- [ ] Runtime JSON is outside Program Files.
- [ ] Setup marker controls first-run redirection.
- [ ] Test Connection is available.
- [ ] Configuration writes atomically.
- [ ] AppLauncher settings match the web executable.
- [ ] Publish script includes required tools.
- [ ] Smoke test validates the first-run page.

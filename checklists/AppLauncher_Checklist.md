# AppLauncher Checklist

- [ ] Single-instance protection.
- [ ] Configuration parsing validated.
- [ ] Executable and DLL fallback supported.
- [ ] Health endpoint available.
- [ ] Duplicate server prevented.
- [ ] Port conflict distinguished.
- [ ] Browser opens after readiness.
- [ ] Tray icon works.
- [ ] Logs stored outside Program Files.
- [ ] Shutdown command works.
- [ ] Installer invokes graceful shutdown.
- [ ] Launcher published as a single-file executable.

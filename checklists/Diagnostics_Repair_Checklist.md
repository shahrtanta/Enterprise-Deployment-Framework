# Diagnostics and Repair Checklist

- [ ] Diagnostics sanitize errors.
- [ ] Write probes are removed.
- [ ] Network checks use timeouts.
- [ ] Support bundle excludes configuration and data.
- [ ] Repair supports dry-run.
- [ ] Existing configuration is preserved.
- [ ] Customer data is never deleted.
- [ ] Repair report is generated.
- [ ] Exit codes are deterministic.

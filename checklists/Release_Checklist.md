# Release Checklist

## Build
- [ ] Restore succeeds.
- [ ] Release build succeeds.
- [ ] Tests pass.
- [ ] Dependency review completed.

## Publish
- [ ] Main executable or DLL exists.
- [ ] Launcher exists where required.
- [ ] Production configuration exists.
- [ ] Development settings removed.
- [ ] Static assets are local.
- [ ] Diagnostics included.
- [ ] Manifest generated.

## Installer
- [ ] Clean-machine install tested.
- [ ] Offline install tested.
- [ ] First run without database tested.
- [ ] Database setup tested.
- [ ] Upgrade tested.
- [ ] Repair tested.
- [ ] Uninstall preserves data by default.

# Database Standard

## Bootstrap

1. Validate configuration.
2. Test server or file.
3. Create storage.
4. Create or restore database.
5. Apply migrations.
6. Seed.
7. Record schema version.
8. Run health checks.

## Safety

- Backup before destructive migration.
- Validate before restore.
- Sanitize connection strings.
- Use bounded retries.
- Produce migration reports.

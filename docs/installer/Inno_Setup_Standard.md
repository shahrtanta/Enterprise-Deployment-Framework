# Inno Setup Standard

## Required

- Stable AppId.
- Upgrade-safe directory.
- Local prerequisites.
- Optional shortcuts.
- Graceful shutdown.
- Firewall only when required.
- Data preservation by default.
- Installer logging.
- UTF-8 source.

## Prohibited

- Internet downloads in `[Run]`.
- Blind deletion of customer data.
- Direct shortcut to a web host when a launcher is required.
- Firewall rules for unused ports.
